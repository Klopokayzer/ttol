Интернет-магазин мастерской колец.

Идея

Создать сайт в формате интернет-магазин для мастерской колец. Организовать основной раздел «каталог», в котором будет выкладываться продукция мастерской с фото, описанием, ценами. Предусмотреть возможность владельцам мастерской добавлять новые позиции. Клиенты будут оформлять заказы с помощью раздела «корзина». С помощью личного кабинета клиенты смогут отслеживать статус заказа. Для владельца мастерской предусмотреть административный доступ для отслеживания заказов.
